Desktops FileSync

Cross-platform LAN file sync desktop app. Pick a folder, name your device — files sync automatically across every machine on your network. No cloud, no accounts, no configuration files.

▶ How to Run ↗ GitHub
Features
📡
UDP Peer Discovery
Devices on the same LAN find each other automatically via UDP broadcast every 5 seconds — no IP addresses to configure.
⚡
WebSocket File Transfer
Files are pushed over a persistent WebSocket connection (port 9876) the moment they change. Deletions propagate too.
👁️
Folder Watcher
Chokidar watches your chosen folder in real time. New, modified, and deleted files are detected and synced instantly.
🔁
Catch-up Sync
On reconnect, each peer requests the other's full file list. Missed changes while a device was offline are automatically filled in.
🖥️
System Tray
Runs silently in the menu bar / system tray. Closing the window keeps it alive. Quit only when you mean to.
📋
Activity Log
The Dashboard shows a live stream of peer connections, file sends, receives, and deletions with timestamps — persisted in SQLite.
🚀
Launch at Login
Enable auto-launch in Settings and FileSync starts with your machine — syncing begins before you open a browser.
🔒
100% Local
Everything stays on your LAN. No relay servers, no cloud uploads, no telemetry. Your files never leave your network.
How to Run
⚙️
Getting Started
1
Requirements
Node.js 18 or later and npm. On macOS you may also need Xcode Command Line Tools for native modules.
2
Install dependencies
npm install
3
Start in development mode
npm run dev # Electron window opens with hot-reload via electron-vite
The app window opens automatically. Go to Settings, choose a watch folder and give your device a name, then save. Repeat on every machine you want to sync.
4
Build a distributable
# macOS (Apple Silicon .dmg) npm run dist:mac # macOS 15 — LAN privacy workaround build npm run dist:mac15 # Windows (NSIS installer .exe) npm run dist:win # Linux (.AppImage) npm run dist:linux
Platform Command Output Arch
🍎 macOS dist:mac dist/*.dmg arm64
🍎 macOS 15 dist:mac15 dist/*.dmg x64 (Intel)
🪟 Windows dist:win dist/*.exe x64
🐧 Linux dist:linux dist/*.AppImage x64
5
Sync across devices
Run FileSync on two or more machines on the same Wi-Fi or LAN. They discover each other automatically — no IP addresses needed. Drop a file in the watched folder on one machine and it appears on the others within seconds.
